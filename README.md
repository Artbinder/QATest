This is an end to end QA testing platform for the ArtBinder web application. We are using playwright

The Examples folder contains screenshots with context for components of the testing platform.
The file Examples/Prompt_Template.png gives context for how to structure testing prompts for another application
The file Examples/Test_Hooks.png gives context for how to set up testing for another application
The files Examples/Sample_Test1.png and Examples/Sample_Test2.png give an example of an automated test for another application

The Source Code folder contains all context for the ArtBinder web application including webpage code and the source code for the whole web application 
The folder Source Code/xylor contains the source code for the application written in Ruby
The folder Source Code/webpages contains the webpage code taken from the web browser

The Files folder contains any excel files or image files that the tests need in order to import into ArtBinder.

The Test List folder contains an excel file with all of the recorded end to end test cases with directions for each
The file Test List/CMS Production Regression Test - 2.xlsx contains the excel file with all recorded test cases

None of the testing platform has been coded so far

We want to fully automate the testing process and run it as a pipeline
